تثبيت واستخدام نظام إدارة عيادة ابتسامة المشاهير لتجميل الاسنان
==============================================

متطلبات النظام:
--------------
1. نظام تشغيل Windows
2. Python 3.6 أو أحدث
3. المكتبات المطلوبة (Flask, SQLAlchemy, وغيرها)

طريقة التثبيت:
------------
1. قم بتثبيت Python من الموقع الرسمي: https://www.python.org/downloads/
2. قم بتثبيت المكتبات المطلوبة باستخدام الأمر التالي:
   pip install flask flask-sqlalchemy

طريقة تشغيل البرنامج:
-------------------
1. الطريقة الأولى (من سطح المكتب):
   - قم بتشغيل ملف "create_shortcut.vbs" بالنقر عليه مرتين
   - سيتم إنشاء اختصار على سطح المكتب باسم "ابتسامة المشاهير"
   - انقر على الاختصار لتشغيل البرنامج

2. الطريقة الثانية (من مجلد البرنامج):
   - قم بتشغيل ملف "start_dental_clinic.bat" بالنقر عليه مرتين

3. الطريقة الثالثة (للمطورين):
   - افتح موجه الأوامر (Command Prompt)
   - انتقل إلى مجلد البرنامج
   - اكتب الأمر: python app.py

بعد تشغيل البرنامج:
-----------------
- سيتم فتح المتصفح تلقائياً على العنوان: http://127.0.0.1:5000
- إذا لم يتم فتح المتصفح تلقائياً، يمكنك فتحه يدوياً وكتابة العنوان: http://127.0.0.1:5000

ملاحظات هامة:
-----------
- لا تغلق نافذة موجه الأوامر أثناء استخدام البرنامج
- لإيقاف البرنامج، اضغط على Ctrl+C في نافذة موجه الأوامر أو أغلق النافذة
